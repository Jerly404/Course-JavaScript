Uso de la consola en JavaScript
  La consola es una herramienta de depuración que permite ver la salida del código,
  inspeccionar variables y detectar errores.

Comandos básicos de console
  1. Imprimir mensajes en la consola
    console.log("This is a message")
    console.info("This is information")
    console.warn("This is a warning")
    console.error("This is a error")

      - log     -> mesaje normal 
      - info    -> Información 
      - warn    -> Advertencia en amarrillo
      - error   -> mensaje de error en rojo

  2. Mostrar varibales en la consola
    let name = "Jerly";
    let age = "9";
    console.log("Name:", name);
    console.log("Age", age);

  3. Mostrar objetos y arreglos
    let person = { name:"Jerly", age: 9 }; // Display the object in a table
    let numbers = [1, 2, 3, 4, 5];  // Display values in table 

    console.table(person);
    console.table(numbers);
  
  4. Medir tiempo de ejecución
    console.time("Run time")
    for (let i = 0; i < 100000; i ++) {}
    console.timeEnd("Run time")
