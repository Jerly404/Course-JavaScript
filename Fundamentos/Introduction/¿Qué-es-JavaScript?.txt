¿Qué es JavaScript?
  JavaScript(JS) es un lenguaje de progrmación de alto nivel dieñado inicalmente
  para hacer que las páginas web fueran interactivas. Hoy en día, JS es un lenguaje
  multipropósito que se usa tanto en el frontend (navegadores) como en el backend
  (servidores con Node.js).

Características clave
  - Interpretado: Se ejecuta en tiempo real sin nesesidad de compilación.
  - Basado en eventos: Puede reaccionar a un evento como clics, desplazamiento y teclado 
  - Orientado a objetos: Soporta el paradigma orientado a objetos basados en prototipos
  - Dinámico: No nesesitas definir tipos de datos antes de usar variables.
  - Multiplataforma: Funciona en cualquier sistema operativo con un navegador o servidores con Node.js 

¿Dónde se usa JavaScript?
  - Desarollo web: Interactividad en páginas web.
  - backend (servidor): Con Node.js se puede manejar bases de datos, servidores, APIs, etc.
  - Desarollo móvil: Con frameworks como React Native.
  - Desarollo de juegos: Motores como Phaser.js o en Godot con GDScipt similar a JS.
  - Aplicaciones de escritorio: Como Electron.js(VS Code está hecho con esto).
